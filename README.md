# Bitácora de Opciones (React + Vite + Tailwind)

App simple para registrar operaciones (CSP/CC/IC), calcular BE y resultado neto, con dashboard básico.

## Comandos
```bash
npm install
npm run dev       # desarrollo
npm run build     # producción
npm run preview   # previsualizar build
```

## Deploy en Vercel
1. Crea cuenta en https://vercel.com
2. Sube el repo (GitHub) o usa "Import Project" y arrastra esta carpeta.
3. Vercel detecta Vite y publica tu sitio en `https://<tu-app>.vercel.app`.
